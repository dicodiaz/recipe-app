# 📖 Recipe App <a name="about-project"></a>

**Recipe App** keeps track of all your recipes, ingredients and inventory. It allows you to:

- Save Ingredients
- Keep track of what you have
- Create recipes
- Generate a shopping list

## 💻 Getting Started <a name="getting-started"></a>

To run the project locally, follow these steps.

### Prerequisites

In order to run this project you need to:

- Install [Git](https://git-scm.com/)
- Install [Node](https://nodejs.org/en/)
- Install a code editor, We suggest [VsCode](https://code.visualstudio.com/)

### Setup

Clone the repo into your local machine:

```bash
git clone https://github.com/dicodiaz/recipe-app
```

Install the dependencies using bundler:
```bash
bundle install
```

### Usage

To start you can open your terminal and in the folder write
```bash
ruby main.rb
```
